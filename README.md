# 🏥 Hospital Management System

A full-featured Hospital Management System developed using Python Flask and MySQL. This project helps hospitals manage patients, doctors, appointments, prescriptions, billing, and reports through a centralized dashboard.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Admin Login
- Dashboard Analytics
- Manage Patients
- Manage Doctors
- Appointment Management
- Prescription Management
- Billing System
- Reports Dashboard
- Change Password

### 👨‍⚕️ Doctor Module
- Doctor Login
- Doctor Dashboard
- View Appointments
- Add Prescriptions
- Patient Details
- Change Password

### 🧑‍🤝‍🧑 Patient Module
- Patient Registration
- Patient Login
- Forgot Password
- Patient Dashboard
- View Profile
- Appointment Booking
- View Prescriptions
- View Bills
- Change Password

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sumitpatel24/Hospital-Management-System.git
```

### Move To Project Folder

```bash
cd Hospital-Management-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
